# PHP DB (MySQL) Demo

## Setup
- Clone Repo and move demo_db folder to Ampps\www 
- Build MySQL DB using sql\db_create_and_seed.sql
- Ensure Apache Server is started (Ampps)
- Load [http://localhost/demo_db/](http://localhost/demo_db/) in your Browser
